# Images TROD angine pour la démo (demo-app-animation.html)

La séquence TROD angine affiche, au clic sur chaque badge, une capture (JPG) dans la fenêtre sous la barre, comme pour Imbruvica.

## Fichiers JPG à placer dans ce dossier (`website/demo/`)

À partir des URLs du raw outils_metier (ligne TROD angine), enregistrer ou capturer chaque page en **JPG** et nommer les fichiers ainsi :

| Fichier JPG | URL source (raw outils_metier) |
|-------------|--------------------------------|
| **trod-compte-rendu.jpg** | https://www.ameli.fr/finistere/content/compte-rendu-prise-en-charge-de-l-angine-l-officine |
| **trod-logigramme-ameli.jpg** | https://www.ameli.fr/finistere/content/prise-en-charge-de-l-angine-l-officine |
| **trod-reco-has-enfant.jpg** | https://www.has-sante.fr/upload/docs/application/pdf/2016-11/v1-fm_rhino-angine_enfant_cd-171116.pdf |
| **trod-reco-has-adulte.jpg** | https://www.has-sante.fr/upload/docs/application/pdf/2016-11/v1-fm_rhino-angine_adulte_cd-171116.pdf |
| **trod-bon-prise-en-charge-adulte.jpg** | https://www.offibox.fr/_files/ugd/a6f194_27069eb3981b4773979e82af49a676b0.pdf |
| **trod-bon-prise-en-charge-enfant.jpg** | https://www.offibox.fr/_files/ugd/a6f194_bb08e13497094e41b70b5a942b3da220.pdf |

Pour les PDF (HAS, Offibox) : ouvrir le PDF, faire une capture d’écran de la première page (ou exporter en image), enregistrer en JPG avec le nom indiqué.

Une fois les 6 JPG placés dans ce dossier, la démo affichera chaque image dans le panneau sous la barre lors du clic sur le badge correspondant.
